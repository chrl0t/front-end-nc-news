## Run The App Locally:

In your terminal:

Check your version of Node.js - the minimum version of Node required to run locally is v14.8.0:

### `node --version`

Clone into the repo:

### `git clone https://github.com/chrl0t/front-end-nc-news`

### `cd front-end-nc-news`

Install the dependencies:

### `npm install`

Run the app:

### `npm start`
